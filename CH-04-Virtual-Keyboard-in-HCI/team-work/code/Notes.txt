Prerequisites:
- Python 3.7 to 3.13 (Python 3.14 is not yet supported by Kivy)
- pip (Python package manager)
> Important:
    Kivy does not currently have wheels built for Python 3.14. If you have Python 3.14 installed, you'll need to downgrade to Python 3.11, 3.12, or 3.13 to use this application.

Troubleshooting:
1- Issue: Kivy installation fails with "No matching distribution found for kivy_deps.sdl2_dev"
- Cause: You are using Python 3.14 or later, which is not yet supported by Kivy.
- Solution: Downgrade Python to version 3.11, 3.12, or 3.13
- Steps:
    1- Check current version: python --version
    2- winget install Python.Python.3.13 -e
    3- py -3.13 --version
    4- py -0
    5- py -3.13 -m pip install kivy
